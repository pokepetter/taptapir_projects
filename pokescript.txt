## A Programming Language for the Web

• Can be run in web browsers.
• Inspired by Python's syntax.
• Transpiles into JavaScript.
• List comprehension.


• Range.
```
for i in 8:
    print(i)
```

• Enumerate.
```
for i, e in enumerate(my_list):
    print(i, e)
```

• List Comprehension.
```
my_list = [1, 0, 9, 4, 8]
my_list = [e for e in my_list if e < 5]
print(my_list)
>>> [1, 0, 4]
```
